Pearson's chi-squared test is used to determine whether there is a statistically significant difference between the expected frequencies and the observed frequencies in one or more categories of a contingency table.
Observations are classified into mutually exclusive classes and the test is useful to assess wheter observations consisting of measures on two variables are independent of each other.
The null hypothesis states that the occurence of the two outcomes is statistically independent. Each observation is allocated on one cell of a two dimensional contingency table according to the values of the outcomes.

After seeing the plots we will apply Shapiro-Wilk test to our data. This tests aims to verify the null hypothesis (Data on each group is normal). If the p-values we obtain are smaller than our significance level, we will reject the null hytpothesis. We can see on the third row, a boolean variable stating if in each treatment we can consider data to be normal (accept null hypothesis) or we should reject it (H1 hypothesis).
